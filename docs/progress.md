# Progress

## Current status

Working static skeleton. The game runs end to end in the browser.

## What already works

- Start screen and screen switching.
- Randomized route selection from the riddle pool.
- Treasure map shows the selected route in order for a configurable time.
- Wind transition hides the map.
- Per-island multiple-choice questions; correct answers advance.
- Win and lose screens; lose screen shows progress.
- Debug mode (answer reveal + skip button).

## Partially working

- CSS animations are basic and could be polished.

## Not implemented yet

- Real images (using emoji/text placeholders for now).
- Sound effects.

## Known limitations

- Map cannot be reopened once hidden (by design).
- A single wrong answer ends the game (by design).
- Content is a small Hebrew riddle pool.

## Current prototype goal

Confirm the full game loop is solid with placeholders, keeping the project fully static.
